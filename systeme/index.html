Programmation système

La programmation système est un type de programmation qui vise au développement de programmes qui font partie du système d'exploitation d'un ordinateur ou qui en réalisent les fonctions.

mini shell

Dans le cadre du cursus j'ai eu l'occasion de recréer en duo une base de shell en C (minishell), qui peut gérer des pipes, redirections, variable d'environnement, plusieurs built in, signaux...

HTTP server

Toujours dans le cadre du cursus mais cette fois optionnel et en C++ j'ai refait en duo un serveur HTTP (HTTP-serveur), c'est a dire entre autre: gestion de sockets, multiplexing I/O, gestions des codes d'erreur, parsing HTTP, respect de la norme RFC... Inspiré de nginx.
